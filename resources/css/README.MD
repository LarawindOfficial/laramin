# Classes

## Buttons
1. Button
```html
<!-- Default Button -->
<button class="button">Button</button>
```
2. Variants
```html
<!-- Primary Button -->
<button class="button button-primary">Button</button>
<!-- Secondary Button -->
<button class="button button-secondary">Button</button>
```

3. Action
```html
<button class="button create-button">Create</button>
```
